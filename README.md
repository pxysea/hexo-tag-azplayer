# hexo-tag-azplayer

## 安装
进入 hexo 主目录，然后执行
```
npm i -save  "https://github.com/pxysea/hexo-tag-azplayer.git"
```
## 配置

## 使用示例
B站视频示例
```javascript
{% azplayer  https://www.bilibili.com/video/BV1QS421K7mD/ %}
```
Youtube 示例
```javascript
{% azplayer  https://www.youtube.com/watch?v=xxxxxxxx '标题' %}
```
